# 📦 Inventory Management System

## 📌 Project Overview
This is a Java console-based Inventory Management System that allows users to manage products using CRUD operations. The system is connected to a MySQL database using JDBC.

---

## 🚀 Features
- Add new products
- View all products
- Update product details
- Delete products
- Database connectivity using JDBC

---

## 🛠 Technologies Used
- Java
- JDBC
- MySQL
- OOP Concepts
- Exception Handling

---

## 🏗 Project Structure
InventoryJDBC/
│── Main.java  
│── DBConnection.java  
│── Product.java  
│── ProductDAO.java  

---

## 💻 How to Run

1. Clone the repository
   git clone https://github.com/Abinaya8146/InventoryJDBC.git

2. Open in IDE (VS Code / IntelliJ)

3. Configure MySQL database

4. Compile and Run
   javac Main.java
   java Main
